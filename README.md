# Bookcast

PDFドキュメントをポッドキャスト形式の音声ファイルに変換するWebアプリケーションです。

## 概要

BookcastはPDFファイルをアップロードし、AI技術を使用してポッドキャスト形式の音声ファイルを自動生成するシステムです。PDF内容の理解、台本作成、音声合成までを統合して処理します。

## アーキテクチャ

本プロジェクトはフロントエンドとバックエンドで構成されています：

### Frontend (Streamlit)
- PDFアップロードとプロジェクト管理インターフェース
- チャプター境界設定用PDFビューアー
- 処理状況監視と音声ファイルダウンロード機能

### Backend (FastAPI)
- PDF処理とAI統合のAPIエンドポイント
- Gemini APIによるOCRとテキスト抽出
- AI台本生成とテキスト音声変換（TTS）
- Google Cloud Storageとの連携
